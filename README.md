# Floppa Kivy — Мессенджер для локальной сети

Простой консольный мессенджер для обмена сообщениями в пределах одной локальной сети. Работает по протоколу UDP, автоматически определяет локальный IP‑адрес и позволяет отправлять/принимать сообщения между устройствами.
Пожалуйста! **Используйте это программу!**

## Особенности

- **Автоматическое определение локального IP‑адреса**.
- **Отсутствие графического интерфейса** (работа в консоли).
- **Мгновенный обмен сообщениями** в локальной сети.
- **Простота запуска** — не требует настройки сервера.
- **Мультиплатформенность** (работает на Windows, Linux, macOS при наличии Python).

## Системные требования

- Доступ к локальной сети (Wi‑Fi или Ethernet).
- Открытые UDP‑порты (по умолчанию — порт `5000`).

## Установка и запуск

1. Скопируйте всю папку проекта на нужное устройство.
2. Откройте терминал в директории проекта.
3. Запустите приложение:

   ```bash
   python main.py
   ```

## config.py
   ```pythone
   import socket

# Порт для обмена сообщениями
PORT = 5000

# Время ожидания ответа (сек)
TIMEOUT = 2

# Размер буфера для приёма сообщений
BUFFER_SIZE = 1024

# Получаем локальный IP-адрес
def get_local_ip():
    try:
        # Создаём UDP-сокет, чтобы узнать локальный IP
        sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
        sock.connect(("8.8.8.8", 80))  # Google DNS
        local_ip = sock.getsockname()[0]
        sock.close()
        return local_ip
    except Exception:
        return "127.0.0.1"

LOCAL_IP = get_local_ip()

   ```
Параметры можно изменить (по желанию)


## Ограничения

- Работает только в локальной сети (не через интернет).
- Использует UDP — возможны потери пакетов (маловероятно для локальной сети).
- Не поддерживает отправку файлов.
